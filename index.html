For the visualization I chose to represent the distribution of license statuses using a line chart. For the x axis I used Temporal(T) and Quantitative (Q) for the y axis. I used interactivity with a dropdown selection menu to filter the data by license type. This helps the visualization be more clear by allowing the user to isolate and study the growth or decline of a specific license category. For the first visualization I chose to use a bar chart with the encoding types Nomial(N) for Liscense status on the x axis and Quantitative(Q) for the count on the Y axis. I used the tableau color scheme to the Liscense status variable to show a clear visual distinction.
